2. システム概要

2.1 システムの目的

体重と身長を入力してBMIを算出・表示するWebアプリケーションを構築する。ユーザー認証、履歴保存、推移グラフ表示の機能を備える。

2.2 システム構成

本アプリケーションはクライアント・サーバー構成とする。フロントエンドはHTML・CSS・JavaScriptで構成し、バックエンドでユーザー認証とデータ永続化を行う。

4.2 FNC-001: ログイン

処理概要

ログインボタン押下時に、入力されたユーザー名・パスワードで認証を行う。

処理仕様

  • ユーザー名・パスワードをサーバーに送信し認証を行う
  • 認証成功時: セッションを生成し、BMI算出画面（SCR-002）に遷移する
  • 認証失敗時: エラーメッセージ「ユーザー名またはパスワードが正しくありません」を表示する
  • パスワードはハッシュ化して保存・照合する

4.3 FNC-002: ログアウト

処理概要

ログアウトボタン押下時に、セッションを破棄してログイン画面に遷移する。

処理仕様

  • サーバーサイドのセッションを破棄する
  • ログイン画面（SCR-001）にリダイレクトする

4.4 FNC-003: 入力バリデーション

処理概要

各画面での入力値の妥当性を検証する。

ログイン画面バリデーション

優先順検証対象検証内容エラーメッセージ
1ユーザー名未入力チェックユーザー名を入力してください
2パスワード未入力チェックパスワードを入力してください

BMI算出画面バリデーション

優先順検証対象検証内容エラーメッセージ
1日付未入力チェック日付を入力してください
2日付日付形式チェック正しい日付を入力してください
3身長未入力チェック身長を入力してください
4体重未入力チェック体重を入力してください
5身長数値チェック身長は数値で入力してください
6体重数値チェック体重は数値で入力してください
7身長正の数値チェック（> 0）身長は正の数値で入力してください
8体重正の数値チェック（> 0）体重は正の数値で入力してください

処理仕様

  • バリデーションは優先順に従い上から順に実行する
  • すべてのバリデーションエラーを同時に表示する
  • エラーがある場合、後続処理は実行しない

4.5 FNC-004: BMI算出

処理概要

入力された身長と体重からBMI値を算出する。

計算式

BMI = weight / (height / 100)²
変数内容単位
weight入力された体重kg
height入力された身長cm

処理仕様

  • 身長をcmからmに変換する（÷100）
  • BMI値を算出する
  • 小数点以下第2位を四捨五入し、第1位まで表示する

4.6 FNC-005: BMI判定

処理概要

算出されたBMI値をもとに、日本肥満学会基準に基づく判定を行う。

判定ロジック

if (BMI < 18.5) → "低体重（やせ）"
else if (BMI < 25.0) → "普通体重"
else if (BMI < 30.0) → "肥満（1度）"
else if (BMI < 35.0) → "肥満（2度）"
else if (BMI < 40.0) → "肥満（3度）"
else → "肥満（4度）"

4.7 FNC-006: 履歴保存

処理概要

BMI算出結果を日付・ユーザーIDとともにデータベースに保存する。

処理仕様

  • ログイン中のユーザーIDを取得する
  • 入力された日付、身長、体重、算出されたBMI値を保存する
  • 同一ユーザー・同一日付のデータが既存の場合は上書き更新（UPSERT）する
  • 保存成功後、履歴一覧を再読込して表示を更新する

4.8 FNC-007: 履歴一覧表示

処理概要

ログインユーザーの過去の記録を一覧表示する。

処理仕様

  • ログインユーザーの全履歴をデータベースから取得する
  • 日付の降順（新しい順）でテーブルに表示する
  • 各行に削除ボタンを配置する

4.9 FNC-008: 履歴削除

処理概要

選択した履歴レコードを削除する。

処理仕様

  • 削除ボタン押下時に確認ダイアログを表示する
  • 確認後、該当レコードをデータベースから削除する
  • 削除後、履歴一覧を再読込して表示を更新する

4.10 FNC-009: グラフ表示

処理概要

過去6か月間の体重・BMI値の推移を折れ線グラフで描画する。

処理仕様

  • 現在日付から過去6か月間のデータをデータベースから取得する
  • Chart.jsを使用して折れ線グラフを描画する
  • X軸: 日付（時系列）
  • Y軸（左）: 体重（kg）
  • Y軸（右）: BMI値
  • データポイント間は直線で接続する
  • データがない日付はプロットしない
