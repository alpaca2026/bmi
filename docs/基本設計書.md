## 2. システム概要

### 2.1 システムの目的

体重と身長を入力してBMIを算出・表示するWebアプリケーションを構築する。

### 2.2 システム構成

本アプリケーションはクライアントサイドのみで動作するシングルページアプリケーション（SPA）とする。サーバーサイド処理は不要であり、HTML・CSS・JavaScriptのみで構成する。

```
┌─────────────────────────────────┐
│          ブラウザ（クライアント）       │
│                                 │
│  ┌───────────┐  ┌───────────┐  │
│  │  HTML     │  │   CSS     │  │
│  │ (画面構造) │  │ (スタイル) │  │
│  └───────────┘  └───────────┘  │
│  ┌───────────────────────────┐  │
│  │       JavaScript          │  │
│  │  ・入力バリデーション        │  │
│  │  ・BMI算出ロジック          │  │
│  │  ・判定処理                │  │
│  │  ・結果表示制御             │  │
│  └───────────────────────────┘  │
└─────────────────────────────────┘
```

### 2.3 技術スタック

| 項目 | 技術 |
|------|------|
| マークアップ | HTML5 |
| スタイル | CSS3 |
| ロジック | JavaScript（ES6+） |
| 動作環境 | モダンブラウザ（Chrome, Firefox, Safari, Edge） |

## 4. 機能設計

### 4.1 機能一覧

| 機能ID | 機能名 | 概要 |
|--------|--------|------|
| FNC-001 | 入力バリデーション | 身長・体重の入力値を検証する |
| FNC-002 | BMI算出 | 身長と体重からBMI値を算出する |
| FNC-003 | BMI判定 | BMI値から肥満度判定を行う |
| FNC-004 | 結果表示 | BMI値と判定結果を画面に表示する |

### 4.2 FNC-001: 入力バリデーション

#### 処理概要

算出ボタン押下時に、入力値の妥当性を検証する。

#### バリデーションルール

| 優先順 | 検証対象 | 検証内容 | エラーメッセージ |
|--------|----------|----------|------------------|
| 1 | 身長 | 未入力チェック | 身長を入力してください |
| 2 | 体重 | 未入力チェック | 体重を入力してください |
| 3 | 身長 | 数値チェック | 身長は数値で入力してください |
| 4 | 体重 | 数値チェック | 体重は数値で入力してください |
| 5 | 身長 | 正の数値チェック（> 0） | 身長は正の数値で入力してください |
| 6 | 体重 | 正の数値チェック（> 0） | 体重は正の数値で入力してください |

#### 処理仕様

- バリデーションは優先順に従い上から順に実行する
- エラーが検出された場合、該当するエラーメッセージをエラー表示エリアに表示する
- すべてのバリデーションエラーを同時に表示する
- エラーがある場合、BMI算出処理は実行しない

### 4.3 FNC-002: BMI算出

#### 処理概要

入力された身長と体重からBMI値を算出する。

#### 計算式

```
BMI = weight / (height / 100)²
```

| 変数 | 内容 | 単位 |
|------|------|------|
| weight | 入力された体重 | kg |
| height | 入力された身長 | cm |

#### 処理仕様

- 身長をcmからmに変換する（÷100）
- BMI値を算出する
- 小数点以下第2位を四捨五入し、第1位まで表示する

### 4.4 FNC-003: BMI判定

#### 処理概要

算出されたBMI値をもとに、日本肥満学会基準に基づく判定を行う。

#### 判定ロジック

```
if (BMI < 18.5) → "低体重（やせ）"
else if (BMI < 25.0) → "普通体重"
else if (BMI < 30.0) → "肥満（1度）"
else if (BMI < 35.0) → "肥満（2度）"
else if (BMI < 40.0) → "肥満（3度）"
else → "肥満（4度）"
```

### 4.5 FNC-004: 結果表示

#### 処理概要

BMI算出・判定の結果を画面に表示する。

#### 処理仕様

- エラー表示エリアを非表示にする
- 結果表示エリアを表示状態にする
- BMI値を「あなたのBMI: XX.X」形式で表示する
- 判定結果を「判定: XXXXXXXXX」形式で表示する

## 6. データ設計

### 6.1 入力データ

| データ名 | 型 | 範囲 | 説明 |
|----------|------|------|------|
| height | number | 0より大きい正の数値 | 身長（cm） |
| weight | number | 0より大きい正の数値 | 体重（kg） |

### 6.2 出力データ

| データ名 | 型 | 範囲 | 説明 |
|----------|------|------|------|
| bmi | number | 小数点以下第1位 | 算出されたBMI値 |
| category | string | 判定基準に基づく文字列 | BMI判定結果 |

### 6.3 定数定義

| 定数名 | 値 | 説明 |
|--------|------|------|
| BMI_UNDERWEIGHT | 18.5 | 低体重の上限値 |
| BMI_NORMAL | 25.0 | 普通体重の上限値 |
| BMI_OBESE_1 | 30.0 | 肥満（1度）の上限値 |
| BMI_OBESE_2 | 35.0 | 肥満（2度）の上限値 |
| BMI_OBESE_3 | 40.0 | 肥満（3度）の上限値 |

## 7. エラー処理設計

### 7.1 エラー処理方針

| エラー種別 | 処理方法 |
|------------|----------|
| 入力バリデーションエラー | エラー表示エリアにメッセージを表示し、BMI算出処理を中止する |
| 予期しないエラー | コンソールにエラーを出力し、ユーザーに汎用エラーメッセージを表示する |

### 7.2 エラーメッセージ一覧

| エラーID | メッセージ | 発生条件 |
|----------|----------|----------|
| ERR-001 | 身長を入力してください | 身長が未入力 |
| ERR-002 | 体重を入力してください | 体重が未入力 |
| ERR-003 | 身長は数値で入力してください | 身長が数値でない |
| ERR-004 | 体重は数値で入力してください | 体重が数値でない |
| ERR-005 | 身長は正の数値で入力してください | 身長が0以下 |
| ERR-006 | 体重は正の数値で入力してください | 体重が0以下 |
