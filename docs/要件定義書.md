## 2. プロジェクト概要

### 2.1 目的

体重と身長を入力し、BMI（Body Mass Index）を算出・表示するアプリケーションを開発する。ユーザーごとに日々の記録を保存し、過去の推移をグラフで確認できる機能を提供する。

### 2.2 背景

BMIは肥満度を表す指標として国際的に広く用いられており、健康管理の基本的な指標である。本アプリケーションは、ユーザーが簡単にBMIを確認できる手段を提供するとともに、継続的な健康管理を支援する。

### 2.3 スコープ

- ログイン機能の提供
- BMI算出機能の提供
- 入力値に対するバリデーション
- BMI判定結果の表示
- 日々の体重・身長・BMIの履歴保存機能
- 過去6か月間の推移グラフ表示機能

## 3. 機能要件

### 3.1 ログイン機能

#### 3.1.1 概要

ユーザー名とパスワードを入力してシステムにログインする。

#### 3.1.2 入力項目

| 項目 | データ型 | 必須 | 備考 |
|------|----------|------|------|
| ユーザー名 | 文字列 | ○ | 半角英数字 |
| パスワード | 文字列 | ○ | 8文字以上 |

#### 3.1.3 機能仕様

- ユーザー名とパスワードが一致する場合、ログインを許可する
- 認証失敗時はエラーメッセージを表示する
- ログイン成功後、BMI算出画面に遷移する
- ログアウト機能を提供する

### 3.2 BMI算出機能

#### 3.2.1 概要

ユーザーが日付・体重・身長を入力すると、BMI値を算出して表示する。

#### 3.2.2 入力項目

| 項目 | データ型 | 単位 | 必須 | 備考 |
|------|----------|------|------|------|
| 日付 | 日付 | - | ○ | デフォルトは当日。過去日付の入力も可能 |
| 身長 | 数値 | cm | ○ | 正の数値であること |
| 体重 | 数値 | kg | ○ | 正の数値であること |

#### 3.2.3 出力項目

| 項目 | データ型 | 備考 |
|------|----------|------|
| BMI値 | 数値 | 小数点以下第1位まで表示 |
| 判定 | 文字列 | BMI値に基づく肥満度判定 |

#### 3.2.4 BMI計算式

```
BMI = 体重(kg) ÷ (身長(m))²
```

※ 身長はcmで入力されるため、計算時にmへ変換する（÷100）。

#### 3.2.5 BMI判定基準（日本肥満学会基準）

| BMI値 | 判定 |
|-------|------|
| 18.5未満 | 低体重（やせ） |
| 18.5以上 25.0未満 | 普通体重 |
| 25.0以上 30.0未満 | 肥満（1度） |
| 30.0以上 35.0未満 | 肥満（2度） |
| 35.0以上 40.0未満 | 肥満（3度） |
| 40.0以上 | 肥満（4度） |

### 3.3 履歴保存機能

#### 3.3.1 概要

BMI算出時に、日付・体重・身長・BMI値をユーザーごとに保存する。

#### 3.3.2 保存項目

| 項目 | データ型 | 備考 |
|------|----------|------|
| ユーザーID | 文字列 | ログインユーザーを識別 |
| 日付 | 日付 | 記録対象の日付 |
| 身長 | 数値（cm） | 入力された身長 |
| 体重 | 数値（kg） | 入力された体重 |
| BMI値 | 数値 | 算出されたBMI値 |

#### 3.3.3 機能仕様

- 同一日付のデータが既に存在する場合は上書き更新する
- 履歴は日付の降順で一覧表示する
- 履歴の削除機能を提供する

### 3.4 グラフ表示機能

#### 3.4.1 概要

過去6か月間の体重・BMI値の推移を折れ線グラフで表示する。

#### 3.4.2 グラフ仕様

| 項目 | 内容 |
|------|------|
| 表示期間 | 過去6か月間 |
| X軸 | 日付 |
| Y軸（左） | 体重（kg） |
| Y軸（右） | BMI値 |
| グラフ種類 | 折れ線グラフ |

#### 3.4.3 機能仕様

- 過去6か月間のデータをもとにグラフを描画する
- データがない日付は表示しない（プロットしない）
- 体重とBMI値を同一グラフ上に表示する（2軸）

### 3.5 入力バリデーション

| 条件 | エラーメッセージ |
|------|------------------|
| ユーザー名が未入力 | ユーザー名を入力してください |
| パスワードが未入力 | パスワードを入力してください |
| 認証失敗 | ユーザー名またはパスワードが正しくありません |
| 日付が未入力 | 日付を入力してください |
| 日付が不正 | 正しい日付を入力してください |
| 身長が未入力 | 身長を入力してください |
| 体重が未入力 | 体重を入力してください |
| 身長が数値でない | 身長は数値で入力してください |
| 体重が数値でない | 体重は数値で入力してください |
| 身長が0以下 | 身長は正の数値で入力してください |
| 体重が0以下 | 体重は正の数値で入力してください |

## 5. 画面構成

### 5.1 画面一覧

| 画面名 | 概要 |
|--------|------|
| ログイン画面 | ユーザー名・パスワードを入力してログインする |
| BMI算出画面 | 日付・身長・体重を入力しBMIを算出する。履歴一覧を表示する |
| グラフ画面 | 過去6か月間の体重・BMI推移をグラフ表示する |

### 5.2 ログイン画面

#### 画面要素

1. **タイトル**: アプリケーション名の表示
2. **ユーザー名入力欄**: ユーザー名を入力するフィールド
3. **パスワード入力欄**: パスワードを入力するフィールド
4. **ログインボタン**: ログイン処理を実行するボタン
5. **エラー表示エリア**: 認証エラーを表示するエリア

### 5.3 BMI算出画面

#### 画面要素

1. **ヘッダー**: ログインユーザー名、ログアウトボタン、グラフ画面へのリンク
2. **日付入力欄**: 記録対象の日付を入力するフィールド（デフォルト: 当日）
3. **身長入力欄**: 身長を入力するフィールド（単位: cm）
4. **体重入力欄**: 体重を入力するフィールド（単位: kg）
5. **算出ボタン**: BMIを算出し履歴に保存するボタン
6. **結果表示エリア**: BMI値と判定結果を表示するエリア
7. **エラー表示エリア**: バリデーションエラーを表示するエリア
8. **履歴一覧**: 過去の記録を日付降順で一覧表示するテーブル

### 5.4 グラフ画面

#### 画面要素

1. **ヘッダー**: ログインユーザー名、ログアウトボタン、BMI算出画面へのリンク
2. **グラフ表示エリア**: 過去6か月間の推移グラフを表示するエリア
3. **凡例**: 体重・BMI値の凡例表示

## 6. 用語定義

| 用語 | 説明 |
|------|------|
| BMI | Body Mass Index（ボディマス指数）。体重と身長から算出される肥満度の指標。 |
| 日本肥満学会基準 | 日本肥満学会が定めるBMIに基づく肥満度の判定基準。 |
| 履歴 | ユーザーが日々記録した身長・体重・BMI値のデータ。 |
