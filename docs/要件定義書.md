## 2. プロジェクト概要

### 2.1 目的

体重と身長を入力し、BMI（Body Mass Index）を算出・表示するアプリケーションを開発する。

### 2.2 背景

BMIは肥満度を表す指標として国際的に広く用いられており、健康管理の基本的な指標である。本アプリケーションは、ユーザーが簡単にBMIを確認できる手段を提供する。

### 2.3 スコープ

- BMI算出機能の提供
- 入力値に対するバリデーション
- BMI判定結果の表示

## 3. 機能要件

### 3.1 BMI算出機能

#### 3.1.1 概要

ユーザーが体重と身長を入力すると、BMI値を算出して表示する。

#### 3.1.2 入力項目

| 項目 | データ型 | 単位 | 必須 | 備考 |
|------|----------|------|------|------|
| 身長 | 数値 | cm | ○ | 正の数値であること |
| 体重 | 数値 | kg | ○ | 正の数値であること |

#### 3.1.3 出力項目

| 項目 | データ型 | 備考 |
|------|----------|------|
| BMI値 | 数値 | 小数点以下第1位まで表示 |
| 判定 | 文字列 | BMI値に基づく肥満度判定 |

#### 3.1.4 BMI計算式

```
BMI = 体重(kg) ÷ (身長(m))²
```

※ 身長はcmで入力されるため、計算時にmへ変換する（÷100）。

#### 3.1.5 BMI判定基準（日本肥満学会基準）

| BMI値 | 判定 |
|-------|------|
| 18.5未満 | 低体重（やせ） |
| 18.5以上 25.0未満 | 普通体重 |
| 25.0以上 30.0未満 | 肥満（1度） |
| 30.0以上 35.0未満 | 肥満（2度） |
| 35.0以上 40.0未満 | 肥満（3度） |
| 40.0以上 | 肥満（4度） |

### 3.2 入力バリデーション

| 条件 | エラーメッセージ |
|------|------------------|
| 身長が未入力 | 身長を入力してください |
| 体重が未入力 | 体重を入力してください |
| 身長が数値でない | 身長は数値で入力してください |
| 体重が数値でない | 体重は数値で入力してください |
| 身長が0以下 | 身長は正の数値で入力してください |
| 体重が0以下 | 体重は正の数値で入力してください |

## 5. 画面構成

### 5.1 メイン画面

本アプリケーションは単一画面で構成される。

#### 画面要素

1. **タイトル**: アプリケーション名の表示
2. **身長入力欄**: 身長を入力するフィールド（単位: cm）
3. **体重入力欄**: 体重を入力するフィールド（単位: kg）
4. **算出ボタン**: BMIを算出するためのボタン
5. **結果表示エリア**: BMI値と判定結果を表示するエリア
6. **エラー表示エリア**: バリデーションエラーを表示するエリア
